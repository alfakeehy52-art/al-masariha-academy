# المرحلة E — واجهة الزائر + سوشيال «قريباً»

## ما نُفّذ

1. **سوشيال «قريباً»** — في تذييل كل الصفحات العامة (`shared-layout.js`) + صفحة **تواصل معنا**.
2. **رسائل آمنة** — `js/visitor-message.js` يخفي أخطاء تقنية (Supabase، RLS، SQL…) عن الزائر.
3. **صفحات مربوطة:** انضمام، متابعة طلب، استكمال مرفقات، تواصل، متجر، متابعة طلب متجر.

## اختبار سريع

| # | ماذا |
|---|------|
| 1 | افتح أي صفحة عامة → تذييل **تابعنا** → اضغط أيقونة سوشيال → تظهر «قريباً» |
| 2 | صفحة **تواصل** → نفس الأيقونات في العمود الجانبي |
| 3 | (اختياري) افصل النت مؤقتاً وأرسل طلب انضمام → الرسالة **بدون** كلمات تقنية |

## ما لم يُنفَّذ (مؤجّل)

- ربط روابط واتساب/إنستغرام/تيك توك الحقيقية من الإعدادات (مرحلة الهوية الرسمية).
