# المرحلة C — صلاحيات الفريق (دليل المالك)

## ماذا تفعل مرة واحدة في Supabase

نفّذ **`docs/STAFF_PANEL_ROLES.sql`** (انسخه من الملف أو من رسالة المطور).

## إضافة موظف جديد للوحة

1. سجّل الكادر في **الكوادر (موحّد)** ببريد إلكتروني.
2. أضف بريده في **`supabase-config.js`** → `adminEmails` ثم ارفع الموقع (أو اطلب من المطور).
3. أنشئ له حساب دخول في **Supabase → Authentication** (نفس البريد).
4. افتح **صلاحيات الدخول** → اختر الدور → احفظ.
5. الموظف: خروج ثم دخول من جديد.

## معنى الأدوار (القائمة الجانبية)

| الدور | ماذا يرى |
|--------|-----------|
| **المدير العام** | الكل |
| **مدير عمليات** | بدون «النظام» (إعدادات/صلاحيات) |
| **مشرف** | مثل المدير تقريباً بدون «النظام» |
| **موظف** | الطلبات والمراجعة فقط |
| **مدرب** | تشغيل محدود (مباريات/إحصائيات…) |
| **مشاهدة فقط** | عرض محدود |
