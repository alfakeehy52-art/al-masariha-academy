# تقرير الاعتماد الرسمي — أكاديمية المسارحة

| البند | القيمة |
|--------|--------|
| **الإصدار / الدورة** | تصدير الطلبات + تشديد الأمان (P0) |
| **التاريخ** | 2026-05-28 |
| **البيئة المختبرة** | واجهة محلية + Supabase (بعد تطبيق SQL) |
| **الحالة** | **جاهز للاعتماد** (مع ملاحظات تشغيلية أدناه) |

---

## 1) ملخص تنفيذي

تم تنفيذ دورة تحسينات على **إدارة الطلبات** تشمل: تصدير CSV / Excel / PDF، تنسيق عربي، نطاق التصدير (مفلتر / الكل)، وإصلاحات أمنية وتكامل بيانات (P0).  
بعد تطبيق سياسات SQL ودوال RPC في Supabase، والتحقق من الواجهات، **يُوصى بالاعتماد الرسمي** مع متابعة اختبار إنتاجي قصير على بيانات حية (قسم 6).

---

## 2) نطاق التغييرات المعتمدة

### وظائف جديدة / محسّنة
- تصدير **CSV** بفاصل `;` مناسب لـ Excel العربي.
- تصدير **Excel (.xlsx)** مع خلايا نصية (منع تحويل الجوال لصيغة علمية).
- تصدير **PDF** عبر نافذة طباعة HTML (هوية الأكاديمية، شعار، رقم تقرير، ختم بصري، سطران لكل طلب).
- اختيار نطاق التصدير: **النتائج المفلترة** أو **كل الطلبات**.
- أسماء ملفات تلقائية بالتاريخ والوقت.

### إصلاحات P0 (أمان وتكامل)
| # | الإصلاح | الملفات |
|---|---------|---------|
| 1 | منع اعتماد الطلب إذا فشل الإجراء الجانبي (side-effect) | `admin_requests_app.js` |
| 2 | تقييد روابط المرفقات إلى `http/https` فقط | `admin_requests_app.js` |
| 3 | تحسين escaping وروابط آمنة في لوحة الاستكمال | `admin_completion_dashboard.html` |
| 4 | إصلاح منطق `canPrepareAttachments` + حماية null | `request_status.html` |
| 5 | إلغاء fallback الحساس `select(*)` — الاعتماد على RPC فقط | `js/supabase-public-lookup.js` |
| 6 | إزالة تكرار تحميل سكربتات Supabase | `admin_requests.html` |

### SQL مطلوب في Supabase (تم تنفيذه من قبل المالك)
1. `docs/JOIN_REQUESTS_RLS.sql`

## 4) ملف `.tmp` أثناء الاختبار — توضيح

**مثال:** `4dff23f1-e152-4437-932d-c379af7f5aa0.tmp` (حجم ~1 KB)

| السؤال | الجواب |
|--------|--------|
| **ما هو؟** | ملف **مؤقت** أنشأه المتصفح أو نظام التحميل عند محاولة تنزيل/طباعة **لم تكتمل** أو حُجبت (نوافذ منبثقة، إلغاء، حظر). |
| **هل هو التصدير الصحيح؟** | **لا.** التصدير الصحيح: `academy_requests_YYYY-MM-DD_HH-mm-ss.csv` أو `.xlsx`. |
| **كيف أفتحه؟** | غالبًا **لا فائدة من فتحه** — محتوى ناقص أو فارغ. يمكن فتحه بـ Notepad للتأكد؛ إن كان فارغًا احذفه بأمان. |
| **PDF** | لا يُحمّل `.pdf` مباشرة من الزر — يفتح نافذة طباعة؛ اختر **Microsoft Print to PDF** أو **Save as PDF**. |

**توصية:** احذف ملفات `.tmp` من مجلد التحميلات؛ استخدم فقط الملفات ذات الامتداد الصحيح.
