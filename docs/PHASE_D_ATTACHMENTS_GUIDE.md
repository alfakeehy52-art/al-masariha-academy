# المرحلة D — مراجعة المرفقات (دليل المالك)

## أين تراجع؟

| الصفحة | الاستخدام |
|--------|-----------|
| **إدارة الطلبات** (`admin_requests.html`) | فتح أي طلب → قسم «مراجعة واعتماد المرفقات» |
| **طلبات حسب النوع** (ولي أمر، داعم، كادر…) | نفس المودال مع تفاصيل النوع |
| **استكمال الطلبات** (`admin_completion_dashboard.html`) | نظرة سريعة + اعتماد نهائي |

## أنواع الطلب

| النوع | مرفقات |
|--------|--------|
| لاعب | نعم — كاملة |
| ولي أمر | نعم — حسب الغرض (جديد / ربط) |
| كادر / مدرب | نعم — + سائق باص إن وُجد |
| داعم | نعم — شعار إلزامي للراعي المحتمل |
| عضو أكاديمية | **لا** — قبول بعد مراجعة البيانات فقط |

## قبل «قبول الطلب»

1. افتح الطلب → راجع كل ملف (معاينة / قبول / رفض).
2. يجب **قبول كل المرفقات الإجبارية** قبل الاعتماد (ما عدا عضوية الأكاديمية).
3. إن نقص ملف: «طلب استكمال» ثم يكمل المتقدم من `request_completion.html`.

## اختبار سريع

- طلب ولي أمر تجريبي → قبول الملفات → قبول الطلب.
- طلب داعم → تحقق من عقد + تعهد.
- عضوية → لا يظهر panel مرفقات → قبول مباشر بعد التحقق.
